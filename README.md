## Features
|Feature|Use|
|----|-----|
|whoami|Find current user|
|reverse shell|Execute commands|
|tasklist|List tasks|
|LOLBas methods|Bypass detection|
|ls|List files|
|cat|Read files|
|exit|exit server or/and client|
